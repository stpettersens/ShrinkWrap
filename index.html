What is ShrinkWrap?

ShrinkWrap is a collection of C++ code to provide a system native executable wrapping around your Java or other VM-based application or similar.

ShrinkWrap is provided as a thin SDK, offering data encoding of the host application resources, to use with your C++ compiler of choice.

You can clone ShrinkWrap from its Git repo, using: 
git clone git://github.com/stpettersens/ShrinkWrap

Compatibility

ShrinkWrap overall should be mostly compatible with most C++ compilers and has been tested with and is known to work with the following:

  • GNU G++
